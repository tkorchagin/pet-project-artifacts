# GentleBreath Pet Project - Presentation Materials

Этот репозиторий содержит материалы презентаций и артефакты для пет-проекта GentleBreath.

## Содержание

### Презентации
- `index.html` - Основная презентация
- `ideas.html` - Презентация идей проекта
- `ai-intro.html` - Введение в AI компоненты
- `drive.html` - Презентация драйва проекта
- `research.html` - Исследовательские материалы
- `pack.html` - Пакет материалов
- `v1.html`, `v2.html`, `v3.html` - Различные версии презентаций

### Документация
- `pet-landing.md` - Описание лендинга проекта
- `sources.md` - Источники и ссылки

### Медиа
- `photo.jpeg`, `photo2.jpeg` - Фотографии для презентаций
- `qr.png` - QR-код

## История изменений

Все изменения отслеживаются через Git commits. Используйте понятные commit messages для удобной навигации по истории.

## Структура

Все файлы хранятся в корне репозитория для удобного доступа во время презентаций.
